# Subclub.eu subtitle download

Downloads the selected subtitles.

## Installation

Global installation:

```npm i subclub-eu -g```

## Usage

Run command ```subclub```

Fill the questions:

```bash
[?] Movie name: avatar
[?] Select the specific movie: Avatar @2009
[?] Select the download file: Avatar.2009.DVDRip.srt

```
